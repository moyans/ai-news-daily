# AI News Daily

每日AI新闻爬虫数据仓库

## 项目结构

```
.
├── data/           # 每日爬取的新闻数据
├── scripts/        # 爬虫和处理脚本
├── config/         # 配置文件
└── README.md
```

## 数据格式

每日新闻数据保存在 `data/` 目录下，按日期组织：

```
data/
├── 2025-03-21.json
├── 2025-03-22.json
└── ...
```

## 使用方式

1. 运行爬虫脚本获取当日AI新闻
2. 处理数据后保存到日期对应的JSON文件
3. Git提交推送更新

## 数据字段

每条新闻包含：
- 标题
- 来源
- 链接
- 发布时间
- 摘要
- 分类标签
