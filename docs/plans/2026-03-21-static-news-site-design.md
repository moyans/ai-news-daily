# AI News Daily 静态新闻站点设计文档

## 概述

基于 Git 的纯静态个人新闻展示站点，部署到 GitHub Pages。

## 需求

- **纯静态展示**：类似博客，只读
- **技术栈**：HTML + CSS + 轻量 JS
- **部署**：GitHub Pages
- **功能**：分类筛选、日期归档、搜索
- **视觉风格**：极简（黑白配色、清晰排版、大量留白、极简卡片）

## 技术方案

选择**单页应用**方案：

- HTML + JSON + 轻量 JS
- 构建脚本将 Markdown 转为 JSON 数据
- 前端 JS 实现筛选/搜索
- 轻量、简单、易维护

## 文件结构

```
ai-news-daily/
├── docs/                      # GitHub Pages 发布目录
│   ├── index.html           # 主页面
│   ├── style.css           # 极简风格样式
│   ├── app.js              # 筛选/搜索逻辑（~5KB）
│   └── data.json           # 所有新闻数据
├── scripts/
│   └── build.py            # 构建脚本：Markdown → JSON
├── data/                    # 原始 Markdown（保留）
│   └── YYYY-MM-DD/source/content.md
└── docs/plans/              # 设计文档
```

## 功能实现

### 分类筛选
- 点击分类标签，JS 过滤 `data.json` 重新渲染
- 支持 URL hash 定位（如 `#category=智能体`）

### 搜索功能
- 前端 JS 全文搜索标题+摘要
- 实时搜索，无防抖（数据量小）

### 日期归档
- 侧边栏月份选择
- 点击跳转到对应月份的新闻

## 视觉风格

- **字体**：`-apple-system, "Segoe UI", sans-serif`
- **配色**：
  - 背景：`#fff`
  - 文字：`#1a1a1a`
  - 次级文字：`#666`
  - 边框：`#e5e5e5`
  - 链接/高亮：`#0066cc`
- **卡片**：`1px #e5e5e5` 边框，`8px` 圆角，`hover` 微阴影
- **间距**：`24px` 卡片间距，`16px` 内边距
- **响应式**：移动端单列，桌面端最大宽度 `800px`

## 构建流程

1. `scripts/build.py` 扫描 `data/` 目录下的所有 Markdown 文件
2. 解析 Markdown 提取结构化数据
3. 合并所有数据生成 `docs/data.json`
4. 复制静态文件到 `docs/` 目录
5. GitHub Pages 从 `docs/` 目录发布

## 部署

GitHub Pages 配置：
- Source: `docs/` folder from `main` branch
- Custom domain: 可选
- HTTPS: 强制启用
